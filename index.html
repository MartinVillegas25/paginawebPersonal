SKILLS



HTML

HTML 5 es la quinta revisión importante del lenguaje básico de la World Wide Web, un estándar que sirve para definir la estructura y el contenido de una página Web.

CSS

CSS es un lenguaje de diseño gráfico que permite definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

Github

GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones, un portafolio de codigos

Bootstrap

Bootstrap es un kit de herramientas de código abierto para desarrollos web responsive con HTML, CSS y JavaScript. Con él puedes darle forma a tu sitio web a través del uso de sus librerías CSS y JavaScript.
